# HTML

## HyperText Markup Language

Web上で公開される文書の論理構造、レイアウト情報、および、他の文書やデータとのリンク関係などを定義するために用いられる標準的な書式のこと。

お堅い説明をしてみたが，発生原理がWeb上での文書構造を決めたもののため，本来はデザイン云々とかは苦手な分野。なので，要素を重ねたり，横に並べたりなんてことは本質的に苦手だったりする。

マークアップと表現されたりする。

Webのフロントを仕事にしたければ避けては通れない道。

### コードサンプル
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>サンプルページ</title>
  <!-- Webページのデザインはcssファイルを読み込んで行いましょう -->
  <link rel="stylesheet" href="./css/style.css">
</head>
<body>
  <div>
    <h1>見出しにはh1～h6のタグを使用します</h1>
    <p>pタグは段落を表現します。<span>文章の範囲はspanタグで指定します</span></p>
    改行はbrタグを使用します。<br>
    divは情報を一塊にして扱うために存在します。divそのものには意味は存在しません。
  </div>
  <script>
    // 最近は読み込み速度を上げるために，bodyタグの最後でスクリプトの読み込みをすることが多いです
  </script>
</body>
</html>
```
